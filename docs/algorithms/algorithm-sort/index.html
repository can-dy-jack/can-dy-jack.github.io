算法之排序

排序就是将一组对象按照某种逻辑顺序重新排列的过程。

排序在商业数据处理和现代科学计算中有着重要的地位，它能够应用于事物处理、组合优化、天体物理学、分子动力学、语言学、基因组学、天气预报和很多其他领域。其中一种排序算法甚至被誉为20世纪科学和工程领域的十大算法之一。
