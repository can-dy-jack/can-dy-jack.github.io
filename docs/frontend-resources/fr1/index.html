New in Git: switch and restore​

该文介绍了 git 新增的两个命令，git switch 用来切换分支，git restore 用来恢复原样，以及解开对 git checkout 的困惑。

info

These methods are still marked experimental, but for all intents and purposes they are here to stay so by all means I encourage everyone to start using them since they will probably make a lot more sense in your head and also it will make git just a little bit less confusing to new users.

Why Apple Will Switch to ARM-Based Macs​

August 26th, 2014 的文章，准确预测到 Mac 电脑会改用 ARM 架构 CPU。

摘要

I’d bet that there are features Apple envisions for the Mac that simply can’t be built while Intel designs the chips inside of them. To implement those ideas, Apple would need to switch the Mac to ARM-based processors, because only then would the company have the ability to design chips customized for specific features. If Apple moved the Mac to ARM-based chips, the company would literally be able to create better products than it can today.

工具​

据意查句​

来自清华大学计算机系自然语言处理实验室（THUNLP）。 WantQuotes主要解决找句子的问题。

关于

早在一年半之前就启动了WantQuotes的研发，我们花费了大量时间收集、清洗数据，同时围绕名句的上下文推荐和语义理解及匹配开展了深入研究，分别提出了新的算法，在评测实验中获得了比现有其他方法更好的效果，并且已有一篇论文发表在本领域顶级会议。🏅

汉语反向词典​

中、英文查词工具，也是来自清华大学计算机系自然语言处理实验室（THUNLP）。

主要功能
  • 输入描述，查找符合描述意思的词语
  • 始终输出100个词语
    • 词语背景颜色的深浅代表模型计算得到的相关度的大小
    • 很多情况下，排序靠后的词语往往和输入描述并不相关
    • 点击词语会弹出浮动窗口，可以查看其基本定义，同时可以点击复制该词语
