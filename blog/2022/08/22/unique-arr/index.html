Javascript数组去重的几种方法

August 22, 2022 · 4 min read
kart jim
本博客创造者

这里一共总结了 方法：

  1. 利用Set
  2. 双层循环判断是否相等,splice()去掉重复值
  3. 利用indexOf去重
  4. 利用includes()
  5. 利用循环加 hasOwnProperty()
  6. 利用filter
  7. 使用 Map 对象
  8. 使用第三方库 Lodash 、Ramda

测试用数组：

var arr = [
  1, 1, 21, 21,0,
  'true', 'false',true, false, 'false', true, 
  undefined,  null, null, NaN, 'NaN', NaN, 'NaN', false, undefined,
  0, 'a', 'a', 'true', 
  {'name': 'jim', 'age': 20},
  {'age': 20, 'name': 'jim'},
  {'name': 'kart', 'age': 21},
  {'name': 'kart', 'age': 21},
  {}, {}
];

利用Set​

function uniqueArr1(arr) {
  return [...new Set(arr)];
  // or
  // return Array.from(new Set(arr))
}

缺点：对象无法去重；

可用于只包含基本数据类型的数组去重。

双层循环判断是否相等,splice()去掉重复值​

function uniqueArr2(arr) {
  // 外层循环元素，内层循环时比较值。值相同时，则删去这个值。
  for(let i = 0;i<arr.length;i++) {
    for(let j = i+1;j<arr.length;j++) {
      if(arr[i] === arr[j]) {
        arr.splice(j, 1);
        j--;
      }
    }
  }
  return arr;
}

缺点：

  • NaN 不等于它本身，所以不能去掉重复的NaN
  • 对象无法去重

利用indexOf去重​

function uniqueArr3(arr) {
  if(!Array.isArray(arr)) {
    throw "不是数组！"
    return;
  }
  const newArr = [];
  for(let i = 0;i<arr.length;i++) {
    if(newArr.indexOf(arr[i]) === -1 ) {
      newArr.push(arr[i]);
    }
  }
  return newArr;
}

缺点：

  • 不能去掉重复的NaN
  • 对象无法去重

利用includes()​

function uniqueArr4(arr) {
  if (!Array.isArray(arr)) {
      console.log('type error!')
      return;
  }
  var newArr= [];
  for (var i = 1; i < arr.length; i++) {
      if (!newArr.includes(arr[i])) {
          newArr.push(arr[i]);
      }
  }
  return newArr;
}

或者可以使用 reduce() 方法：

function uniqueArr4(arr) {
  if (!Array.isArray(arr)) {
      console.log('type error!')
      return;
  }
  return arr.reduce((pre, cur) => {
    if(pre.includes(cur)) {
      return pre;
    } else {
      return [...pre, cur];
    }
  },[])
}

缺点：对象无法去重

利用循环加 hasOwnProperty()​

function uniqueArr5(arr) {
  let map = {};
  return arr.filter((item, idx, oldArr) => {
    if(map.hasOwnProperty(typeof item + '' + item)){
      return false;
    } else {
      map[typeof item + '' + item] = true
      return true;
    }
  })
}

缺点：将object类型都去掉了，仅剩一个

利用filter​

function uniqueArr6(arr) {
  return arr.filter((item, idx, oldArr) => {
    return oldArr.indexOf(item) === idx;
  })
}

缺点：对象无法去重，不能识别 NaN 和 ”NaN“

使用 Map 对象​

利用Map对象中不会出现相同的key值

function uniqueArr7(arr) {
  let map = new Map();
  let newArr = [];
  for (let i = 0; i < arr.length; i++) {
    if(!map.has(arr[i])) {
      map.set(arr[i], true); 
      newArr.push(arr[i]);
    }
  }
  return newArr;
}

缺点：对象无法去重

使用第三方库​

  • Lodash
    • _.uniqWith(arr, _.isEqual)
  • Ramda
    • R.uniq(arr)

总结​

以上方法大部分都能将基本数据类型的重复数去掉，但是无法将 object 类型的数据去重。 利用第三方库能完美去重，但是有时为了一个方法去引入一整个库并不是十分优雅的做法。

所以，当要去重的数组里确认没有 object 类型的数据时，以上方法基本任选；但是如果数组内会有对象，那就要考虑引入第三方库或者在函数中判断对象类型，然后做特殊处理。
